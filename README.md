GroceryGo is a localized grocery e-commerce platform that connects customers with nearby retail stores within a 5 km radius. It reduces warehousing and logistics costs, enables real-time inventory management, supports barcode/QR scanning, and integrates secure UPI payments to empower local retailers and improve customer convenience.


## Features
- User login & profile
- Product search and sorting
- Cart and checkout simulation
- UI/UX Design
- Local store selection


## Tech Stack
- HTML5
- CSS3 (Bootstrap)
- JavaScript (Vanilla)
